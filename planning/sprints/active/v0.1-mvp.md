# Sprint: v0.1 MVP

**Started:** 2026-05-18
**Status:** in-progress

## Goal

Ship a usable PyQt6 GUI wrapping `ptouch-print` for the PT-P710BT — text input, live preview, print.

## Done

- `pyproject.toml` (pipx-installable, console_script `pt-p710bt-label-gui`)
- `pt_p710bt_label_gui.ptouch` — typed wrapper: `query_info()`, `render_preview()`, `print_job()`, `PrintJob.argv()`
- `pt_p710bt_label_gui.app` — `LabelGUI`: tape-info panel, multi-line text, font picker, size, alignment, copies, padding, chain/precut/cutmark, image insert, live debounced preview (350 ms), show-command dialog, print button, output log
- Falls back to `--force-tape-width` for offline preview when the printer is unplugged
- Smoke-tested against connected PT-P710BT (12 mm tape, error 0x0000)

## Outstanding

- Save/load label templates (JSON)
- Barcode/QR helper (probably shell out to `qrencode` → PNG → `--image`)
- Persist last-used font/size/align across runs (QSettings)
- udev-rule installer helper (warn if user is not in the right group)
- Packaging: test `pipx install .` end-to-end
- Optional: thread the subprocess calls so a slow render doesn't freeze the UI
